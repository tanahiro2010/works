# Bokuchiの何がいいか
 
1. **Marpが内蔵されている**
   → フロントマターに `marp: true` 一行で即スライド
2. **Tauri + Rust製で軽い**
   → Electronの1/3以下のメモリ使用量
3. **変数システムがある**
   → エンジニア的にこれがかなり好き
4. テーマが内蔵されている
   → 最初のヘッダーで`theme`パラメータを`default`か `gaia`、`uncover`を選択することである程度デザインが適応される。
   ちなみにこのスライドは `{{theme}}` テーマで作成されてる。

---
 
# 変数システム、具体的には
 
ファイル内で変数を定義できる：
 
```markdown
`<!-- @var name: tanahiro2010 -->`
`<!-- @var date: 今日 -->`
 
# { {name} } の発表資料
作成日: { {date} }
```

(普通に書いたら変数として埋め込まれちゃうのでスペース入れて対策中)
 
グローバル変数はSettings画面で一括管理。
テンプレート運用やチーム共有が楽になる。
