# スライド設計表

作成日: 2026-07-24

## 前提

- 発表時間: 45分
- スライド枚数: 100枚
- 形式: Marp
- テンポ: 1スライド1メッセージの高速紙芝居型
- 対象: OSSに興味はあるけど、まだ踏み出せていない人。特に学生
- 中心メッセージ: OSSは、コードを書く場所である前に、人と繋がって次の挑戦が生まれる場所だ
